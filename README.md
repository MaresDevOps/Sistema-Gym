# Sistema Gym

Este es un panel de administración para un gimnasio diseñado y creado por Mares. 

## Tecnologías Utilizadas
- **Frontend:** HTML, CSS, JavaScript (Vanilla JS, jQuery, Bootstrap 4/5)
- **Backend:** PHP
- **Base de Datos:** MySQL

## Notas de Desarrollo (Problemas Conocidos)
- Hay algunas direcciones (URLs) en el código HTML que podrían requerir correcciones en el futuro. Por ejemplo, enlaces como `<a href="./agregarV.html">` pueden presentar problemas dependiendo de la estructura de navegación actual y deberán ajustarse posteriormente.

## Características
- Gestión de Miembros
- Registro de Visitas
- Gestión de Membresías
- Panel de reportes de ganancias y estadísticas (Chart.js)
